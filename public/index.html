✅ Compte officiel connecté — le token se rafraîchit automatiquement, plus rien à faire.
Connecte le compte du bot une seule fois via l'authentification officielle Kick — le token se renouvellera ensuite automatiquement, sans plus jamais avoir à le faire à la main.
Pour activer le refresh automatique, crée une application sur dev.kick.com et ajoute KICK_CLIENT_ID, KICK_CLIENT_SECRET et KICK_REDIRECT_URI dans les variables Render.
